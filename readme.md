Getting and Cleaning Data Course Project
==

This repo contains all project elements which are:
- Codebook.md - a file containing a detailed description of the dataset, variables and code used;
- run_analysis.r - The R file with the script ready to be run;
- Readme.md - This file you are reading;

Requirements
---
Before running this script you need to:

1. download and unzip the UCI_HAR_Dataset file to your working directory,
2. adjust the stwd funcion inside the R file to define the working directory if its diferent from the default one,
3. install the plyr package,

Other details
---
- The R script was buit using R x64 3.1.0 for windows.
